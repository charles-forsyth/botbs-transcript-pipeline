## About

This project provides text transcripts of the "Back on the Broomstick" podcast episodes. It serves as a resource for listeners who want to read, search, or reference the content of the podcast.

## Usage

The transcripts are provided as plain text files. You can browse the files in this repository or clone the repository to your local machine to access them.

```bash
git clone git@github.com:charles-forsyth/botbs-transcript-pipeline.git
```
